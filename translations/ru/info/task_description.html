Ваша задача - распределить имеющуюся кровь разных типов среди пациентов, нуждающихся в переливании, учитывая ограничения по совместимости каждой группы крови. Запасы крови не всегда достаточны для удовлетворения всех потребностей. Вам будут предоставлены данные о количестве каждой группы крови и потребностях каждого типа пациентов. Ваша цель - оптимально распределить имеющуюся кровь, соблюдая правила совместимости. Имейте в виду, что может быть несколько оптимальных решений - тех, которые используют максимально возможное количество имеющейся крови.

Как это используется: Это задание направлено на эффективное распределение имеющегося количества крови для удовлетворения потребностей пациентов с учетом совместимости. Функция `distribute_blood` рассчитывает соответствующее распределение крови с учетом ограничений совместимости. Выполнив это задание, Вы получите представление об эффективных алгоритмах распределения крови и их практическом применении.
