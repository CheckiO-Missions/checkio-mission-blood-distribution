Ваша місія - розподілити наявну кров різних груп серед пацієнтів, які потребують переливання, враховуючи обмеження сумісності кожної групи крові. Крові не завжди достатньо, щоб задовольнити всі потреби. Вам буде надано інформацію про кількість кожної групи крові та потреби кожного типу пацієнтів. Ваша мета - оптимально розподілити наявну кров, дотримуючись правил сумісності. Пам'ятайте, що оптимальних рішень може бути декілька, тобто таких, що використовують максимально можливу кількість наявної крові.

Обмеження сумісності:

Вхідні дані: Кількість крові(blood_avail) та потреби у крові(blood_needs) для кожної групи крові у вигляді об'єктів карти {% if is_js %}{% else %} словників (dict){% endif %} з рядковими ключами (str{% if is_js %}ing{% endif %}) та цілочисельними значеннями ({% if is_js %}number{% else %}int{% endif %}).

Вихідні дані: На виході буде отримано розподіл кількості крові для кожної групи крові у вигляді {% if is_js %}об'єкта карти об'єктів карти{% else %}словника словників (dict){% endif %} з рядковими ключами (str{% if is_js %}ing{% endif %}) та цілочисельними значеннями ({% if is_js %}number{% else %}int{% endif %}).

Як використовується: Ця місія зосереджена на ефективному розподілі доступної кількості крові для задоволення потреб пацієнтів на основі сумісності. Функція `distribute_blood` обчислює відповідний розподіл крові з урахуванням обмежень сумісності. Виконуючи цю місію, ви отримаєте уявлення про ефективні алгоритми розподілу крові та їх практичне застосування.

Попередні умови
0 ≤ кількість крові;
0 ≤ потреби в крові.
